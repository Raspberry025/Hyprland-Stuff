Hyprland-Stuff

Trying Hyprland for fun lol

This repository contains my personal configuration files and setup for Hyprland, a dynamic tiling Wayland compositor for Linux. The goal of this repository is to explore Hyprland, customize its behavior, and document my configuration as I experiment with different layouts, keybindings, and UI elements.

About

Hyprland is a highly customizable tiling window manager / compositor for Wayland that offers:

Dynamic workspace management

Rich customization options

Support for animations and advanced window behavior
Learn more at the official project: https://github.com/hyprwm/Hyprland

This repository is not an official Hyprland distribution or package.

Repository Contents
├── hyprland.conf        # Main configuration file for Hyprland


The configuration includes personal settings such as:

Keybindings

Workspace definitions

Window behavior and layout rules

Installation

To use this config with Hyprland:

Install Hyprland and dependencies (via your distro’s package manager or from source).

Clone this repository:

git clone https://github.com/Raspberry025/Hyprland-Stuff.git


Copy the config to your local config directory:

mkdir -p ~/.config/hypr
cp Hyprland-Stuff/hyprland.conf ~/.config/hypr/


Restart or start your Hyprland session.
